# Admin Panel 测试计划

## 测试环境

### 环境配置
- **测试地址**: http://localhost:8000/admin
- **登录地址**: http://localhost:8000/admin/login
- **管理员账号**: admin / admin (默认密码)
- **浏览器**: Chrome (推荐使用 Chrome DevTools)

### 启动测试服务器
```bash
# 在项目目录下执行
./venv/bin/python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
