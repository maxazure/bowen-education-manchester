导入说明
CSV 格式要求
  • 文件编码必须为 UTF-8 或 UTF-8 with BOM
  • 第一行必须为表头（字段名）
  • 字段之间用逗号分隔
  • 包含特殊字符的字段用双引号包裹
文章字段说明
  • title - 文章标题（必填）
  • title_en - 英文标题（选填）
  • summary - 文章摘要（选填）
  • content - 文章内容（必填，HTML 格式）
产品字段说明
  • name - 产品名称（必填）
  • name_en - 英文名称（选填）
  • description - 产品描述（必填）
  • price - 价格（选填，数字格式）
