## 📝 使用说明

### 快速开始

#### 1. 查看设计方案
```bash
# 阅读主设计文档
cd docs/ui-design
open README.md
open bootstrap5-design-plan.md
```

#### 2. 预览效果
```bash
# 在浏览器打开模板示例
open bootstrap5-templates-example.html
```

#### 3. 开始开发
```bash
# 按照实施指南操作
open implementation-guide.md

# 创建必要目录
mkdir -p admin/static/css/bootstrap5
mkdir -p admin/static/js/bootstrap5
mkdir -p admin/templates/bootstrap5

# 复制样式文件
cp bootstrap5-styles.css admin/static/css/bootstrap5/admin.css
```
