## 💡 使用建议

### 1. 色彩使用
- 主色（红色）用于主要操作和链接
- 次色（蓝色）用于次要操作
- 功能色用于状态标识
- 中性色用于文本和背景

### 2. 间距使用
- 组件内部使用较小间距（8-16px）
- 组件之间使用中等间距（16-24px）
- 页面边距使用较大间距（24-48px）

### 3. 字体使用
- 标题使用较大字号和较粗字重
- 正文使用标准字号
- 辅助信息使用较小字号和较浅颜色

### 4. 交互反馈
- 所有可点击元素提供悬停效果
- 重要操作提供确认对话框
- 操作结果提供 Toast 提示
- 加载过程显示加载动画
