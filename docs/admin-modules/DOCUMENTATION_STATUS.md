5. 开发步骤(TDD: Red → Green → Refactor)

8. 验证命令
9. 交付物清单
10. 注意事项和相关文档链接

### TODO.md 应包含内容:
1. 模块信息
2. 进行中/已完成/待办事项分类
3. 分阶段的详细待办事项(至少 10-13 个 Phase)
  - Phase 1: 编写测试 (TDD - Red)
  - Phase 2-7: 实现功能 (TDD - Green)
  - Phase 8-9: 模板和静态资源
  - Phase 10: 运行测试验证
  - Phase 11: 代码质量检查 (Refactor)
  - Phase 12: 手动测试
  - Phase 13: 文档与提交
4. 任务统计
5. 完成标准检查清单(详细)
6. 问题与解决记录区
7. 最后更新时间和状态
